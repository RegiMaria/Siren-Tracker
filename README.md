# 🧜 Siren Tracker

> *Tecnologia de ponta para um problema que a ciência ainda não reconhece.*

Plataforma de inteligência naval para monitoramento de entidades aquáticas não-catalogadas. 5 páginas completas, 7 espécies catalogadas, rotas marítimas reais.

---

## 🚀 Setup rápido

```bash
# 1. Clone
git clone https://github.com/seu-usuario/sirenatrack.git
cd sirenatrack

# 2. Ambiente virtual
python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows - Power Shell

# 3. Dependências
pip install -r requirements.txt

# 4. Rode
streamlit run app.py
